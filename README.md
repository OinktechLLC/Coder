## 📖 О проекте

**Coder-Pawno** - это современный веб-редактор кода премиум-класса, вдохновленный VSCode и Pawno, с уникальной возможностью писать код на русском языке! Наша система автоматически переводит русский текст в полноценный код.

### ✨ Основные возможности

- 🔥 **Умный редактор кода** - Monaco Editor (как в VSCode)
- 🇷🇺 **Русский → Код** - Пишите на русском, получайте рабочий код
- ⚡ **Компилятор Pawno** - Полноценная компиляция для Pawn + другие языки
- 🔔 **Push-уведомления** - Мгновенные уведомления для всех пользователей
- 📢 **Рекламные баннеры** - Встроенная система рекламы с canvas
- 🔊 **Звуковые эффекты** - Приятные звуки при действиях
- 💾 **Загрузка/Скачивание** - Работа с проектами локально
- 📱 **Адаптивный дизайн** - Premium дизайн 2026 для ПК и мобильных
- 🎨 **Современный UI** - Градиенты, анимации, glassmorphism

## 🚀 Быстрый старт

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Приложение откроется по адресу: `http://localhost:5173`

### Сборка для продакшена

```bash
npm run build
```

### Предварительный просмотр продакшен сборки

```bash
npm run preview
```

## 🛠 Технологический стек

- **Frontend Framework:** React 18.3+
- **Build Tool:** Vite 6.0+
- **Routing:** React Router DOM 7.1+
- **Editor:** Monaco Editor (@monaco-editor/react)
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Styling:** CSS Variables + Inline Styles
- **Canvas:** HTML5 Canvas API
- **Audio:** Web Audio API

## 🎯 Функционал

### 1. Редактор кода
- Подсветка синтаксиса для Pawno, JavaScript, Python, C++, Java
- Автодополнение и IntelliSense
- Миникарта кода
- Нумерация строк
- Темная тема (VSCode style)

### 2. Компилятор
- **Pawno:** Проверка синтаксиса, поиск ошибок, генерация .amx
- **JavaScript/Python:** Интерпретация кода
- **C++/Java:** Компиляция с выводом результатов

### 3. Русский → Код
Автоматическая конвертация русских ключевых слов:
- `если` → `if`
- `иначе` → `else`
- `пока` → `while`
- `для` → `for`
- `печатать` → `Print`
- `основная` → `main`
- И многие другие...

### 4. Уведомления
- Push-уведомления браузера
- Всплывающие уведомления в интерфейсе
- Звуковые сигналы
- Отправка уведомлений всем пользователям

### 5. Реклама
- Canvas-баннеры в верхней части
- Настройка через панель настроек
- Клик открывает рекламный сайт в новом окне
- Кнопка закрытия
- **Жесткая интеграция** (невозможно удалить без изменения кода)
